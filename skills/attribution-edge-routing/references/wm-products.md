## Other WM products this skill routes to

Verify each product's actual capability via `knowledge-base-ask` before claiming it can answer the user's question. Don't promise features that don't exist.

| **Product** | **What it does** |
|-|-|
| **MBO** (Media Budget Optimizer) | Forecasting + budget scenario optimization. Historical attribution → future allocation. Use for: "forecast", "next quarter / month", "should I shift budget to X", "what if I spend \$Y". |
| **Lift Test** | Geo / audience holdout experiments to measure true incremental impact. Attribution shows calibrated credit; lift tests measure causality. Use for: "real incremental impact", "would this revenue happen without the ad spend", "test if X is working". |
| **Creative Magic** | Asset-level analysis by creative attributes (theme, format, hook). Attribution can group by creative ID; Creative Magic infers semantic attributes. Use for: "which theme works", "which hook converts", "static vs video". |
| **Ads Magic** | Automated bid / budget management on platforms. Prescriptive; attribution is descriptive. Use for: "should I bid higher", "auto-optimize budget", "pause underperformers". |
| **Audience Magic** | Audience generation / segmentation for activation. Attribution shows performance of existing audiences; Audience Magic creates new ones. Use for: "build an audience", "segment my customers", "look-alike from X". |
