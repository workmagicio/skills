# Frequency / granularity default

Pick by time-window size — **don't ask**:

| Window size | Default granularity |
|---|---|
| 1 day | single number (or hourly if supported) |
| ≤ 14 days | daily |
| 15–90 days | weekly |
| > 90 days | monthly |

Mention granularity in one short line only if it might surprise the user ("Showing weekly trend since the window is 30 days — say the word for daily.").
