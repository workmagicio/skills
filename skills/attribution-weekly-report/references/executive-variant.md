## Executive variant (CMO / quarterly)

Triggered by: quarterly cadence OR explicit "for my CMO" / "executive summary". Use `templates/executive-report.md`.

### Differences from default weekly

- **Channel-level only** — strip campaign-level detail
- **Lead with KPI summary** — total spend, total revenue, total ROAS, NC ROAS as a 4-cell header
- **Period-over-period comparison**: quarter vs prior quarter (not WoW)
- **Recommendations should be strategic, not tactical** — "rebalance budget toward iDDA-positive channels next quarter" beats "pause campaign XYZ"
- **Format**: PDF if email delivery; HTML otherwise
- **Length**: 1 page printable

### What an executive-variant report should NOT include

- Campaign-level tables
- Creative-level numbers
- Technical caveats about attribution model methodology (footer line about the model is fine)
- Day-by-day or week-by-week breakdown — quarter is the unit
